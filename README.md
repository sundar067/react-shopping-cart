# 🛒 React Shopping Cart

A fully functional **E-commerce Shopping Cart Application** built using **React**, **Redux Toolkit**, and **Tailwind CSS**. This project demonstrates product listing, cart management, and modern frontend development practices.

---

## 🚀 Features

- 🛍️ Display products in a responsive grid
- ➕ Add items to cart
- ➖ Remove items from cart
- 🔢 Dynamic cart item count
- 💰 Total price calculation
- 🔔 Toast notifications (Add/Remove actions)
- ⚡ Fast and responsive UI
- 🧠 Global state management using Redux Toolkit

---

## 🧰 Tech Stack

- React JS
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- React Hot Toast
